This project is part of the **CodTech Internship** program.

## 📌 Task Description
**Task 4:** Build a movie recommendation system using **Collaborative Filtering** or **Matrix Factorization** techniques.

---

## 🎯 Objective
To develop a system that recommends movies similar to a given movie, based on historical user ratings.

---

## ✅ Approach
- Used the **MovieLens 100k dataset** (`u.data`, `u.item`)
- Merged user ratings with movie titles
- Created a **User-Item Rating Matrix**
- Computed **Cosine Similarity** between items (movies)
- Built a function to recommend top-N most similar movies
